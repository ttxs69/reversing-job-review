# 215. 数组中的第K个最大元素
只想到了暴力解法，直接排序然后取第K个最大的。  
题解提供了两种优化思路：  
1. 快速选择算法，基于快速排序的思想，每次确定一个元素的正确位置，然后利用他的位置和K的关系进行类似二分的查找。
2. 使用最大，或最小堆。
